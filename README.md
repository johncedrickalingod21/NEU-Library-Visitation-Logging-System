# 📚 Library Visitation Logging System

A web-based **Library Visitation Logging System** designed to digitally track and manage student and staff visits using Google authentication and a centralized database.

🔗 **Live Website:** https://library-visit-tracker.lovable.app  

---

## 🚀 Features

- 🔐 **Google Authentication**
  - Users log in using their Google accounts (no manual signup)

- 👤 **Automatic User Registration**
  - Users are automatically added to the database on first login

- 📝 **Visit Logging**
  - Users select their purpose (e.g., Studying, Research, Computer Use)
  - Records visit data with timestamps

- 🔄 **Auto Logout System**
  - Users are automatically logged out after logging a visit

- 📊 **Admin Dashboard**
  - View all visit records
  - Monitor usage trends (daily/monthly)
  - Filter and analyze data

- 🔒 **Secure Data Access (RLS)**
  - Users can only access their own data
  - Admins have full access to all records

---

## 🛠️ Tech Stack

- **Frontend:** Lovable (AI-generated web application)
- **Backend:** Supabase (PostgreSQL + Authentication)
- **Authentication:** Google OAuth
- **Deployment:** Lovable Hosting

---

## 🎯 Project Purpose

This project was developed as a requirement for **Information Management 2**.  

It aims to demonstrate the application of database management concepts, authentication systems, and real-time data handling by creating a functional system that replaces manual library logbooks with a digital solution.

## 📌 Notes

- Admin access is limited to authorized accounts

- The system is still under development and improvements are ongoing
